## 🐲 About the Module

**Pathfinder 2E: Token Pack** is a comprehensive module for **Foundry VTT** that not only adds a massive collection of tokens and art for the **Pathfinder 2e** system but also provides powerful tools to manage your content.

### Key Features
* 🎨 **Massive Collection:** Thousands of ready-to-use tokens for bestiaries, adventure paths, and other rulebooks.
* 🎭 **NPC Disguise:** An advanced tool for changing actor appearances, creating "phases," and managing disguises.
* 🛠️ **GM Utilities:** Features for restoring actors on the scene and fine-tuning compendiums.
* 🖼️ **Character Gallery:** An add-on module with a large library of high-quality art.

---

## 📥 Installation (for Foundry VTT v13)

1.  In the Foundry VTT Setup screen, go to the **Add-on Modules** tab.
2.  Click **"Install Module"**.
3.  In the "Manifest URL" field, paste the following link:
    ### [For Foundry VTT v12](https://github.com/Metofay/pf2e-token-pack/tree/v12)
    ```
    https://raw.githubusercontent.com/Metofay/pf2e-token-pack/v12/module.json
    ```
    ### [For Foundry VTT v13](https://github.com/Metofay/pf2e-token-pack)
    ```
    https://raw.githubusercontent.com/Metofay/pf2e-token-pack/v12/module.json
    ```
4.  Click **"Install"** and wait for the installation to complete.
5.  Activate the module in your game world's settings.

## ⚙️ Module Features

### 1. Compendium Settings
Allows you to check paths for art and tokens, delete unnecessary files, view the number of missing actors, and disable loading of unneeded compendiums to optimize your world's performance.

![Compendium Settings](./data/assets/compendium-settings.png)

### 2. Actor Restoration
Restores actors in the sidebar and on the scene to their original state from the compendium. You can configure exceptions by actor type and folder.

![Actor Restoration](./data/assets/actor-restoration.png)

### 3. NPC Disguise
Allows you to change an actor's appearance, create "phases" (completely separate, editable character sheets), and easily switch between them or revert to the original appearance.

![NPC Disguise](./data/assets/npc-disguise.png)

**Additional functionality:**
* **Quick Access:** Conveniently switch between disguises directly from the token HUD.
* **Effect Triggers:** Automatically apply a disguise when a specific effect is activated on an actor.
* **Flexibility:** The feature works for NPCs as well as other actor types (visual changes only).
* **GM Highlighting:** Tokens with active disguises are highlighted (visible only to the GM), with an option to disable it.
* **Customization:** Control the highlight color, border type, and the position of the HUD element in the settings.
* **For Players:** Allow players to change their own characters' disguises if permitted.

### 4. Add-on Module: Character Gallery
Adds a large, fully localized "Character Gallery" art library for images not found in the standard compendiums.

* **Installation:** Requires installing the separate [**Pathfinder 2E: Token Pack (Character Gallery)**](https://github.com/Metofay/pf2e-token-pack-character-gallery) module.

![Character Gallery](./data/assets/character-gallery.png)
